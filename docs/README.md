## 产品简介

> **分享链接，知识留住。**

从微信公众号、Twitter、微博、博客——你在任何 App 里看到值得保存的内容，分享到 Folio，它就永远是你的了。AI 自动抓取正文、分类、打标签、生成摘要。所有内容存在你的手机上，全文可搜索，离线可读。

**核心体验**：收藏 → 整理 → 找到（三步，零配置）

**设计哲学**：工具美学——安静、高效、有质感

**首发平台**：iOS App

**目标市场**：全球（中英文双语）

## 与时光印记的区别

| 维度 | 时光印记（TECH-003） | Folio（TECH-004） |
|------|---------------------|-------------------|
| 核心体验 | 发现 → 感受 → 分享 | 收藏 → 整理 → 找到 |
| 产品气质 | 感性、温暖、魔法 | 理性、安静、工具美学 |
| 内容来源 | 用户相册照片 | 外部网页/文章 |
| AI 角色 | 叙事作家（写故事） | 图书管理员（分类整理） |
| 用户动作 | 被动发现（打开即有） | 主动收藏（分享链接） |
| 数据存储 | 云端为主 | 本地优先 |
| 目标市场 | 中国市场优先 | 全球市场（中英文） |
