# Folio v3.0 — 交互流程总览

> 从打开 App 到每一个交互分支，完整的层次结构。
> 用于指导 iOS 端实现和后端方案设计。

---

## 一、启动流程

```
App 启动
├── 首次启动 → Onboarding [06-onboarding.html]
│   ├── Page 1: 品牌 — "Folio 记得。然后帮你也记得。"
│   ├── Page 2: 存 — 一键保存 + 洞察摘要预览
│   ├── Page 3: 记 — Echo 回忆预览
│   ├── Page 4: 用 — RAG 问答预览
│   ├── Page 5: 登录
│   │   ├── Apple 登录 → Page 6
│   │   ├── 邮箱登录 → Page 6
│   │   └── 不登录 → Page 6
│   └── Page 6: "开始使用" → Home
│
└── 非首次启动 → Home
```

## 二、Home Feed（主页面，80% 时间）

**原型**: `01-home-feed.html`

```
Home Feed
│
├── 导航栏
│   ├── 标题 "页集"（衬线体）
│   ├── 🔍 搜索图标 → Search & QA [03]
│   └── ⚙️ 设置图标 → Settings [07]
│
├── 日期行 — "三月二十二日，星期六"
│
├── 内容区（可滚动）
│   │
│   ├── 时间分组
│   │   ├── "今天" — section header
│   │   ├── "昨天"
│   │   ├── "本周"
│   │   ├── "上周"
│   │   └── "更早"
│   │
│   ├── Hero 文章（当天第一篇未读）
│   │   ├── 大标题（24pt 衬线体粗体）
│   │   ├── 洞察引文（衬线斜体 + accent 蓝竖线）
│   │   ├── 元信息（来源 · 时间 · 标签）
│   │   └── tap → Reader [04]
│   │
│   ├── 普通文章卡片 ×N
│   │   ├── 未读：标题粗体 + 洞察引文 accent 竖线
│   │   ├── 已读：标题常规 + 洞察引文灰色竖线
│   │   ├── 带缩略图变体（右侧 72×72 图片）
│   │   ├── 收藏标记（★ 右上角）
│   │   ├── tap → Reader [04]
│   │   ├── 右滑 → 切换收藏
│   │   └── 左滑 → 删除
│   │
│   ├── Echo 卡片（穿插在文章之间）
│   │   ├── 居中排版 + 浅灰背景 + "✦ Echo" 标签
│   │   ├── 问题文字（衬线体）
│   │   ├── "揭晓答案" 按钮
│   │   │   └── tap → 原地展开
│   │   │       ├── 答案（衬线粗体 + accent 竖线）
│   │   │       ├── 来源标注
│   │   │       ├── "记得" 按钮 → 收缩为确认行 + 下次时间
│   │   │       └── "忘了" 按钮 → 收缩为确认行 + 下次时间
│   │   └── 三种类型：
│   │       ├── 核心洞察回忆
│   │       ├── 高亮回顾
│   │       └── 关联发现
│   │
│   └── 底部加载更多（三点动画）
│
├── 下拉刷新 → 同步进度条
│
└── 状态变体
    ├── 空状态 — "你读过的，Folio 都记得。" + 粘贴链接按钮
    ├── 首篇收藏 — 单篇文章 + 冷启动引导
    ├── 处理中 — 骨架屏 + 抓取进度条 + 已完成文章
    ├── 离线 — 顶部 banner "离线 · 显示本地内容"
    └── 同步错误 — 顶部 banner + 重试/关闭
```

## 三、Reader（阅读页）

**原型**: `04-reader-full.html`

```
Reader
│
├── 导航栏
│   ├── ← "页集" 返回按钮 → Home [01]
│   └── ⋯ 更多按钮 → Sheet 菜单
│       ├── 收藏
│       ├── 复制 Markdown
│       ├── 阅读偏好
│       ├── 归档
│       ├── 查看原文 → Safari
│       ├── 删除（红色）→ 确认 → Home [01]
│       └── 取消
│
├── 阅读进度条（导航栏下方细线，宽度 = 滚动 %）
│
├── 文章头部
│   ├── 标题（26pt 衬线粗体）
│   └── 元信息（来源 · 日期 · 阅读时间）
│
├── 洞察摘要（可折叠）
│   ├── 收起：✦ + 核心洞察（粗体）+ ∨ 箭头
│   └── 展开：
│       ├── 核心洞察
│       ├── 要点 1/2/3（圆点列表）
│       └── 关联提示 → 相关文章
│
├── 文章正文（Markdown 渲染）
│   ├── 段落（17pt 衬线体，1.75 行距）
│   ├── 二级标题
│   ├── 引用块（左侧灰竖线 + 斜体）
│   ├── 代码（等宽字体 + 灰色背景）
│   └── 高亮交互
│       ├── 选中文字 → 自动高亮（accent 半透明底色）
│       ├── 点击已高亮 → 弹出菜单
│       │   ├── 移除高亮
│       │   └── 复制
│       └── 高亮内容进入知识库，可被 Echo 和 RAG 使用
│
├── 相关收藏（底部）
│   ├── "你的收藏中有 3 篇相关文章"
│   └── 水平滚动卡片 ×3 → tap → Reader（对应文章）
│
├── 底部工具栏
│   ├── 左：🌐 查看原文 → Safari
│   ├── 中：阅读进度 %
│   └── 右：↑ 分享
│
└── 状态变体
    ├── 加载中 — 骨架屏
    ├── AI 处理中 — ✦ "AI 正在阅读这篇文章…"
    ├── 抓取失败 — ⚠️ + 重试 / 查看原文
    └── 内容不可用 — 通用提示 + 查看原文
```

## 四、Search & QA（搜索与问答）

**原型**: `03-search-and-qa.html`

```
Search & QA
│
├── 搜索栏（可输入）
│   ├── 🔍 + 输入框 + ✕ 清除
│   └── "取消" → Home [01]
│
├── 未激活状态
│   └── 提示："搜索，或提问"
│
├── 激活状态（搜索建议）
│   ├── 最近搜索（历史记录）
│   │   └── tap → 直接搜索
│   ├── 建议问句（衬线斜体 + 引号）
│   │   └── tap → 填入并搜索
│   └── 快速操作
│       ├── 粘贴链接 → URL 保存流程
│       └── 记一条笔记 → 笔记保存流程
│
├── 短查询（关键词搜索）
│   ├── "4 篇相关收藏"
│   └── 结果列表（标题 + 摘要 + 关键词高亮）
│       └── tap → Reader [04]
│
├── 长问句（RAG 问答）
│   ├── "✦ 基于 5 篇收藏" 标签
│   ├── 回答正文（衬线体，杂志排版）
│   │   ├── Pull quote（核心观点 + accent 竖线）
│   │   └── 行内引注 ¹ ² ³ → tap → 展开来源
│   ├── 来源文章卡片 ×N
│   │   ├── 标题 + 来源 + 日期
│   │   └── tap → 展开原文摘录 / 再 tap → Reader [04]
│   └── 跟进对话
│       ├── 建议追问 ×2 → tap → 新回答接在下方
│       └── 自由输入 → 回车 → 新回答接在下方
│
├── URL 检测
│   ├── 显示 URL 预览
│   ├── "保存到 Folio" → 保存 + 返回 Home
│   └── "取消" → 返回建议页
│
├── 笔记保存
│   ├── 显示文本预览（衬线斜体）
│   ├── "保存" → 保存 + 返回
│   └── 提示："知识碎片也是知识"
│
└── 无结果
    └── "你的收藏中没有找到与「XX」相关的内容"
```

## 十、转场动画

**原型**: `10-transitions.html`

```
动画规范（Motion tokens）
├── settle  0.4s  spring(bounce 0.05)  — 元素落位
├── quick   0.25s ease                 — 状态反馈
├── ink     0.15s ease-out             — 内容出现
├── exit    0.2s  ease-in              — 元素离开
└── slow    2.0s  linear               — 进度条

7 个核心转场
├── Home → Reader: push 滑入（0.35s）
├── 搜索栏展开: 激活 + 结果淡入（0.25s）
├── Echo 揭晓: 高度扩展 + 答案淡入（settle）
├── AI 摘要展开: 高度 + chevron 旋转（0.35s）
├── RAG 打字机: 逐字出现 + 来源淡入
├── 图片全屏: 缩略图 → 全屏覆盖（0.3s）
└── Sheet 弹出: 底部滑入 + 背景暗化（0.35s）
```
